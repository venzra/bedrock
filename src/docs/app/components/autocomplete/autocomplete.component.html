Autocomplete

This is a more advanced input which can offer potential results from a datasource that can be selected.

Import Module

To use the components here you first need to include this module in your application @NgModule

Basic autocomplete

There are two key attributes to using this component. The search and selection attributes should be methods which return observables. To ensure these return the correct format of data the RockAutocompleteList and RockAutocompleteValue types should be returned.

The following example shows a potential implementation of the methods you could use to manage the autocomplete data, more typically however you may want to use a service and httpClient to load the data via an API.

Display error messages

To display an error works the same as with all other components, no change is required in the javascript code, just add the error component and reference the form control accordingly.
