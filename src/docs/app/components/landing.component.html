Components

Take a look at examples of the various components available and view the associated code to use them.

Form Components

Use these to build your forms in a consistent manner.

Input

The most basic of form fields, used to display several input types `text`, `password` and `textarea` inputs.

Checkbox

Speed up input of binary choices with a checkbox option.

Image

Drag and drop or select an image to attach to an input form.

Action Components

Use these to enable user interactions.

Button

Buttons that can be used to create links, submit forms or enable temporary functionality.

Layout Components

Use these to add some positioning and style to your content.

Alert

Alert messages and blocks to provide warnings and information to users.

Card

A card is used to highlight potentially important information or even to enable a complex action when used with a router link.

Expansion

There are two styles of expansion panel available to show and hide additional or optional content / forms.
